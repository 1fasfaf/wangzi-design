# 上线说明

当前网站是纯静态站，整个 `bedroom-portfolio` 目录可以直接部署。

## 最省事的方式

### Vercel

1. 打开 Vercel
2. 新建项目
3. 上传 `D:\codex\bedroom-portfolio` 整个目录
4. 部署完成后得到公网链接

### Netlify

1. 打开 Netlify
2. 进入 `Add new site`
3. 拖拽上传 `D:\codex\bedroom-portfolio` 整个目录
4. 部署完成后得到公网链接

## 上线前建议

1. 确认 `assets/轮播图/01.jpg` `02.jpg` `03.jpg` 为最终轮播图
2. 检查微信二维码是否正确
3. 如果某些图片过大，先压缩再上传

## 目录

- `index.html` 页面
- `styles.css` 样式
- `script.js` 交互
- `projects.js` 分类数据
- `assets/` 图片资源
